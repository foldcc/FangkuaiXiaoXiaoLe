# 方块消消乐

- Unity 2017.1.0f3
---

- 使用Unity2017开发，实现功能有：
 1. 基本消除功能
 2. UI动画以及代码控制动画
 3. 消除动画以及手势识别
 4. 消除判定、连续消除判定

- UI是自己独立完成比较简单，打包平台我设置的是webGL平台，有需要改成Android的可以自己修改打包设置。
- 该项目供大家参考学习使用，希望大家支持，以后会继续发布自己的一些游戏项目。
